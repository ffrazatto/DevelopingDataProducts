App Objectives

  • Quick explore MTCars data set;
  • Find relations between variables;
  • Simple linear regression model (one regressor);
  • Simple Residue Analysis

Data set: MTCars

  • 32 Observations;
  • 11 Variables;

Since the app objective is to speedup and simplify data exploration, it is recommended that the user first read the variables description and than explore the data or vise versa, although not reading the data set description may lead to hasty conclusions.

R help has a succinct explanation for each variable that can be easily accessed with the following code.

?mtcars
